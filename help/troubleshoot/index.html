トラブルシューティング

よくある問題と解決方法をご案内します。

ログインできません

ログインに関する問題の解決方法をご案内します。

解決方法

  1. メールアドレスとパスワードが正しいか確認してください
  2. パスワードをお忘れの場合は、ログイン画面から「パスワードを忘れた方」をタップして再設定できます
  3. インターネット接続を確認してください
  4. アプリを最新版にアップデートしてください

データが消えてしまいました

データが見つからない場合の対処法です。

解決方法

  1. アカウント登録していれば、クラウドにバックアップされています
  2. 一度ログアウトしてから、再度ログインしてみてください
  3. 複数のアカウントを使用していないか確認してください
  4. アプリを最新版にアップデートしてください

重要：データの紛失を防ぐため、必ずアカウント登録を行ってください。

アプリが起動しない・強制終了する

アプリが正常に動作しない場合の対処法です。

解決方法

  1. アプリを完全に終了してから再起動してください
  2. 端末を再起動してください
  3. OSを最新版にアップデートしてください
  4. アプリを最新版にアップデートしてください
  5. 改善しない場合は、アプリを一度削除して再インストールしてください（アカウント登録していればデータは復元できます）

機種変更後にデータを引き継ぎたい

新しい端末へのデータ移行方法です。

手順

  1. 旧端末でアカウント登録していることを確認（未登録の場合は先に登録してください）
  2. 新しい端末でアプリをダウンロード
  3. 旧端末と同じアカウントでログイン
  4. 自動的にデータが同期されます

ポイント：アカウント登録さえしていれば、いつでもどこでもデータを復元できます。

データが同期されない

複数デバイス間でデータが同期されない場合の対処法です。

解決方法

  1. インターネット接続を確認してください
  2. 両方のデバイスで同じアカウントにログインしているか確認してください
  3. アプリを最新版にアップデートしてください
  4. 一度ログアウトしてから再ログインしてみてください

その他の問題

上記で解決しない場合は、以下の基本的な対処法をお試しください。

  • アプリを最新版にアップデート
  • 端末のOSを最新版にアップデート
  • 端末の空き容量を確保
  • アプリのキャッシュをクリア
  • 端末を再起動

それでも解決しない場合

上記の方法で解決しない場合は、お手数ですが以下の情報を含めてお問い合わせください。

  • ご使用の端末（iPhone、Androidなど）
  • OSのバージョン
  • アプリのバージョン
  • 問題の詳細と発生状況
  • スクリーンショット（可能であれば）

お問い合わせ先：アプリ内の設定画面から「お問い合わせ」を選択してください。
