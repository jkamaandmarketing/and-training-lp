スーパーセットとは？

「スーパーセット」とは、2種目を休憩なしで連続して行うトレーニングテクニックです。特に拮抗筋（反対の動きをする筋肉）を組み合わせることで、時間効率よく両方の筋肉を鍛えることができます。

拮抗筋スーパーセットの例

上腕二頭筋 + 上腕三頭筋
ダンベルカール → すぐにトライセプスプレスダウン
→ 休憩 → 2セット目

胸 + 背中
ベンチプレス → すぐにベントオーバーロウ
→ 休憩 → 2セット目

大腿四頭筋 + ハムストリングス
レッグエクステンション → すぐにレッグカール
→ 休憩 → 2セット目

スーパーセットのメリット

  • 時間短縮：休憩時間を有効活用できる
  • 代謝向上：心拍数が上がりカロリー消費が増える
  • バランス良く鍛えられる：拮抗筋を同時に強化
  • パンプ感の向上：血流が増加する

注意点

  • 器具の確保：2種目分の器具を同時に使えるか確認
  • 混雑時は避ける：ジムが混んでいると器具の独占になりやすい
  • フォームの維持：疲労してもフォームを崩さない

アンドトレーニングでの使い方

スーパーセットをアンドトレーニングで記録する際は、2種目を連続して記録します。

  • メニューにスーパーセットで行う2種目を並べて登録
  • 種目A → 種目B → 種目A → 種目B の順で記録
  • メモ欄に「スーパーセット」と記入して識別
