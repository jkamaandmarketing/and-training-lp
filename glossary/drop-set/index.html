ドロップセットとは？

「ドロップセット」とは、1セットが限界に達した後、休憩を取らずに重量を下げてすぐに続けるトレーニングテクニックです。通常のセットよりも筋肉を深く追い込むことができ、短時間で高強度のトレーニングが可能です。

ドロップセットのやり方

  1. 通常の重量でセットを行い、限界まで追い込む
  2. 休憩を取らずに重量を20〜30%程度下げる
  3. 再び限界まで行う
  4. 必要に応じてさらに重量を下げて繰り返す（2〜3回のドロップが一般的）

ドロップセットの例
ダンベルカール 12kg × 10回（限界）
→ すぐに10kg × 8回（限界）
→ すぐに8kg × 6回（限界）
これで1ドロップセット完了

ドロップセットのメリット

  • 短時間で高強度：時間効率よく筋肉を追い込める
  • 強いパンプ：血流が増加し強いパンプ感が得られる
  • 筋肥大に効果的：代謝ストレスを高められる

注意点

  • 頻繁に使いすぎない：回復に時間がかかるため毎回は避ける
  • フォームを維持：疲労してもフォームを崩さない
  • マシンが向いている：重量変更が素早くできるため
  • 補助種目の最後に：コンパウンド種目よりアイソレーション種目向き

アンドトレーニングでの使い方

ドロップセットもアンドトレーニングで記録できます。

  • 各ドロップを別々のセットとして記録
  • メモ欄に「ドロップセット」と記入して識別
  • ボリューム（総重量）の変化を追跡
